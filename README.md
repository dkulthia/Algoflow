# AlgoFlow – Algorithmic Trading Assistant

## Python (Tkinter, Pandas, yFinance, Matplotlib)

### Developed a desktop application for stock market analysis using algorithmic trading strategies
### Implemented technical indicators (EMA, RSI, Volume Analysis) to generate BUY, SELL, HOLD signals
### Integrated real-time market data with intelligent trade suggestions including stoploss and target levels
### Designed an interactive UI with live search, portfolio tracking, and historical trade logging
